# 可乐爱上课 🎒

可乐（4岁半）的课表打卡与盲盒抽奖小程序，专为小朋友设计。

## ✨ 功能

### 📅 课表管理
- **每周固定排课**：按星期+时间设置重复课程（如每周三、周五）
- **按日期排课**：设置特定日期的临时课程（如补课、特殊活动）
- **多时间段支持**：同一门课可设置多个时间段（如周三17:00和20:00各一节）
- 积分按课程统一计算，不同时间段的同门课共享积分池

### ✅ 打卡积分
- 每次打卡 +10 分
- 100 分为一轮，达到即可抽盲盒
- **首轮福利**：新课程自动赠送 90 分，打卡一次就能抽奖
- 每门课独立积分，互不影响

### 🎁 盲盒抽奖
- 撒花特效 + 音效 + 弹窗动画
- 奖品池可自定义：名称、图标、概率权重
- 支持启用/停用/编辑/删除奖品

### 📆 打卡日历
- 月度视图，颜色标记状态：
  - 🟢 绿色：已打卡
  - 🔴 红色：漏打卡
  - 🟡 黄色：请假
  - ⚪ 灰色：无课程
- 点击日期查看详情
- 支持补打卡 + 备注说明
- 请假功能（不扣分，标记为黄色）

### 🏠 请假标记
- 老师/可乐请假时，标记为请假
- 不扣分、不算漏打卡
- 可备注原因（如"生病""老师请假"）

### ☁️ 云端同步
- 零配置，无需注册
- 设备1复制同步ID → 设备2粘贴 → 数据自动同步
- 不需要服务器

### 📱 移动端优化
- PWA 支持，可添加到手机主屏幕
- 响应式设计，适配手机屏幕
- 导出课程到手机日历（.ics 文件，课前15分钟提醒）
- 数据导入/导出备份

## 🛠 技术栈

- 纯前端：HTML + CSS + JavaScript
- 数据存储：localStorage（本地）+ jsonblob.com（云端同步）
- PWA：Service Worker 离线缓存
- 部署：GitHub Pages

## 📁 项目结构

```
可乐爱上课/
├── index.html          # 主页面
├── manifest.json       # PWA 配置
├── sw.js               # Service Worker
├── css/
│   └── style.css       # 样式
└── js/
    ├── app.js          # 核心业务逻辑
    └── cloud.js        # 云端同步层
```

## 🔄 跨设备同步

1. 设备1：打开 ⚙️ 设置 → ☁️ 云端同步 → 📋 复制同步ID
2. 设备2：打开同一个网址 → ⚙️ 设置 → ☁️ 云端同步 → 粘贴ID → 连接
3. 后续自动同步

## 📦 数据备份

⚙️ 设置 → 📤 导出数据备份 → 下载 JSON 文件
⚙️ 设置 → 📥 导入数据恢复 → 选择 JSON 文件
