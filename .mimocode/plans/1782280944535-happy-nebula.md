# 修复日历请假显示Bug

## 问题描述
请假记录在日历上显示为绿色（已打卡），而不是黄色（请假）。根据README，应该显示为黄色。

## 根本原因
在 `buildDayStatusMap()` 函数中（app.js:1532-1551），逻辑错误：
1. `allChecked` 变量在课程请假时保持为 `true`（因为请假课程不会将其设置为 `false`）
2. 第一个 `if (allChecked)` 分支将状态设置为 `'checked'`（绿色）
3. 第二个 `else if` 分支（处理请假）是死代码，永远不会执行

## 修复方案

### 1. 修复 `buildDayStatusMap()` 函数（app.js:1528-1551）

**当前代码：**
```javascript
const todayCourses = appState.courses.filter(c => 
  c.schedules.some(s => s.dayOfWeek === dayOfWeek)
);

let allChecked = true;
let hasAbsence = false;

todayCourses.forEach(c => {
  const checked = isCheckedIn(c.id, dateStr);
  const isAbsent = appState.checkins.some(chk => 
    chk.courseId === c.id && chk.date === dateStr && chk.isAbsence
  );
  if (!checked && !isAbsent) allChecked = false;
  if (isAbsent) hasAbsence = true;
});

if (allChecked) {
  map[dateStr] = 'checked';
} else if (hasAbsence && !todayCourses.some(c => !isCheckedIn(c.id, dateStr) && !appState.checkins.some(chk => chk.courseId === c.id && chk.date === dateStr && chk.isAbsence))) {
  // 全部课程要么已打卡要么已请假
  map[dateStr] = 'absent';
} else {
  map[dateStr] = 'missed';
}
```

**修复后代码：**
```javascript
const todayCourses = appState.courses.filter(c => 
  c.schedules.some(s => s.dayOfWeek === dayOfWeek) ||
  (c.dateSchedules && c.dateSchedules.some(ds => ds.date === dateStr))
);

let hasChecked = false;
let hasAbsent = false;
let hasMissed = false;

todayCourses.forEach(c => {
  const checked = isCheckedIn(c.id, dateStr);
  const isAbsent = appState.checkins.some(chk => 
    chk.courseId === c.id && chk.date === dateStr && chk.isAbsence
  );
  if (checked) {
    hasChecked = true;
  } else if (isAbsent) {
    hasAbsent = true;
  } else {
    hasMissed = true;
  }
});

if (hasMissed) {
  map[dateStr] = 'missed';
} else if (hasAbsent) {
  map[dateStr] = 'absent';
} else {
  map[dateStr] = 'checked';
}
```

**逻辑说明：**
- 使用三个独立标志：`hasChecked`、`hasAbsent`、`hasMissed`
- 优先级：漏打卡 > 请假 > 已打卡
- 如果有任何课程漏打卡，显示红色
- 如果没有漏打卡但有请假，显示黄色
- 如果全部已打卡，显示绿色

### 2. 修复 `showDayDetail()` 函数（app.js:1566-1568）

**当前代码：**
```javascript
const courses = appState.courses.filter(c => 
  c.schedules.some(s => s.dayOfWeek === dayOfWeek)
);
```

**修复后代码：**
```javascript
const courses = appState.courses.filter(c => 
  c.schedules.some(s => s.dayOfWeek === dayOfWeek) ||
  (c.dateSchedules && c.dateSchedules.some(ds => ds.date === dateStr))

```

**修复内容：**
- 添加对 `dateSchedules` 的检查，确保日期特定课程也会在详情面板中显示
